# FoodSaver - Aplicación Web TFG DAW

**Aplicación de gestión de alimentos para reducir el desperdicio alimentario**

## 🎯 Descripción del Proyecto

FoodSaver es una aplicación web full-stack que permite a los usuarios gestionar sus productos alimenticios, controlar fechas de caducidad y recibir alertas para reducir el desperdicio de comida.

## 🛠️ Stack Tecnológico

### Frontend

- React 18 + Vite
- React Router DOM
- Axios
- CSS Modules

### Backend

- Node.js + Express.js
- JWT para autenticación
- Bcrypt para cifrado de contraseñas
- Sequelize ORM

### Base de Datos

- MySQL 8

## 📁 Estructura del Proyecto

```
foodsaver/
├── frontend/          # Aplicación React
├── backend/           # API REST con Node.js
├── database/          # Scripts SQL
└── README.md
```

## 🚀 Instalación y Ejecución

### Prerrequisitos

- Node.js (v18 o superior)
- MySQL (v8 o superior)
- npm

### Backend

```bash
cd backend
npm install
# Configurar .env con credenciales de base de datos
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 📚 Funcionalidades

- ✅ Registro e inicio de sesión de usuarios
- ✅ Gestión completa de productos (CRUD)

- ✅ Visualización con código de colores por estado
